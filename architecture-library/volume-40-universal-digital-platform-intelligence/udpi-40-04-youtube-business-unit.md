# YouTube Business Unit

Version: draft
Status: Active
Document ID: architecture-library/volume-40-universal-digital-platform-intelligence/udpi-40-04-youtube-business-unit.md

## Purpose
Define YouTube as a platform-managed business unit inside Mariam, not merely a video upload connector.

## Scope
Covers strategy, scripts, titles, descriptions, translations, subtitles, thumbnails, shorts, playlists, publishing, analytics, copyright review, monetization readiness, and policy compliance.

## Chief Agent
YouTube Chief Agent coordinates content strategy, publishing readiness, compliance checks, analytics review, and learning updates.

## Swarm Roles
- Script Agent
- SEO Agent
- Translation Agent
- Thumbnail Review Agent
- Copyright Review Agent
- Publishing Agent
- Analytics Agent
- Growth Agent

## Platform DNA
YouTube DNA stores official API capabilities, publishing limits, metadata rules, content rules, analytics metrics, monetization requirements, and current best practices.

## Compliance Rules
No workflow may use unofficial bypasses. Any automation must respect platform policy, account safety, user consent, and official API limits.

## Execution Flow
Idea -> script -> translation -> thumbnail -> metadata -> compliance review -> human approval -> scheduled publishing -> analytics -> learning update.

- The document contains actionable architecture rules.

- The document can guide implementation without extra interpretation.
